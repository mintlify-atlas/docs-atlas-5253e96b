ServerPilot is a modern Windows desktop application built with Electron and React that turns managing a local Minecraft server into a single-click experience. Whether you're running Paper, Fabric, Forge, or Vanilla, ServerPilot handles server lifecycle, live monitoring, player tracking, file management, and public tunneling via Playit.gg — all from one polished interface.

## What ServerPilot Does

ServerPilot replaces the manual workflow of launching batch files, watching raw terminal output, and editing config files in Notepad. Everything you need to run a local Minecraft server is built into one app:

<CardGroup cols={3}>
  <Card title="One-Click Start/Stop" icon="play">
    Launch your server and Playit.gg tunnel simultaneously. Stop gracefully or force-kill when needed.
  </Card>
  <Card title="Live Monitoring" icon="chart-line">

  </Card>
  <Card title="Integrated Console" icon="terminal">

## Get Up and Running

<Steps>
  <Step title="Install ServerPilot">
    Download `ServerPilot-Setup-X.X.X.exe` from the [Releases page](https://github.com/GKExpo/ServerPilot/releases) and run the installer. No administrator privileges required.
  </Step>
  <Step title="Install Java">
    Modern Minecraft servers require Java 17 or newer. Download [Eclipse Temurin](https://adoptium.net/) (recommended) or Oracle Java, then verify with `java -version` in a terminal.
  </Step>
  <Step title="Download a server JAR">
    Grab the latest [Paper](https://papermc.io/downloads/paper) build for your Minecraft version, place it in a dedicated folder, and rename it to `paper.jar`.
  </Step>
  <Step title="Add your server in ServerPilot">
    Open ServerPilot, click **Add Server**, point it to your server folder and JAR, configure RAM, then click **Start**. The Onboarding Wizard guides you through every field.
  </Step>
</Steps>

<Tip>
  Want friends to join without touching router settings? Follow the [Playit.gg setup guide](/setup/playit) to enable automatic public tunneling in ServerPilot.
</Tip>
